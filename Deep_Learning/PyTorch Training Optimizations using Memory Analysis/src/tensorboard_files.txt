This link contains Pytorch profiler files that can be run on PyTorch Profiler TensorBoard

Steps:
1. Download and move files to ./log/mobilenet_v2 directory
2. Install pyTorch Profiler TensorBoard using terminal command pip install torch_tb_profiler
3. Run profiler using terminal command tensorboard --logdir=./log
